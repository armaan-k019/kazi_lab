# Reviewing own work

Before declaring complete:

1. Re-read the prompt's acceptance criteria. Verify each item is met.
2. Read the files you changed. Look for: em dashes, hardcoded secrets, `any` types without comments, unused imports, marketing voice in comments.
3. Run the application or relevant test. Confirm it works end-to-end, not just that it compiles.
4. Check that you stayed within the prompt's stated scope.
5. If you deviated, document why in the output summary, do not hide it.
6. Verify no commits were made.
